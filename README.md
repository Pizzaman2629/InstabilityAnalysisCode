
# InstabilityAnalysisCode

Set of codes used to analyze different modes for different laser induced perturbations. Designed to interface with Imperial College's Gorgon/Chimera codes. 

---

## Function

### `simulation_analysis.py`
Main class for decoding simulations. Compatible with full vti data or 1D integrated streak data. Also compatible with time dat files dumped from Gorgon/Chimera.

Calculates: 
* Axially integrated density with respect to time. 
* Amplitudes as a function of time for different modes (frequencies).
* Linear regions for each mode (using $r^2$ and polyfit).
* Non-linear saturation times and growth rates for each mode. 
* Shock breakout time.

### `batch_processing.py`
Contains the line sweeper and brane sweeper classes. These classes interface with `simulation_analysis.py` to extract data for multiple simulations. 

* **Line Sweeper:** Takes in simulations that vary along a 1D parameter space. Outputs plot of amplitude dominant mode growth rates as a function of the parameter. 
* **Brane Sweeper:** Extends the line sweeper to a p-dimensional parameter space using corner plots. It is also possible to generate tiled colour coded maps for different parameters.

### Helper Files
* **`main.py`, `reader.py`, `grid.py`:** Helper functions required to read data from Chimera/Gorgon. 
* **`test.py`:** Example usage of the single simulation class in `simulation_analysis.py` or the line sweeper class in `batch_processing.py`.

## Usage

Initialize the single simulation analyzer with your simulation parameters to generate diagnostics:

```python
import simulation_analysis as sa

# Analyze a single simulation and generate a mode plot
sim = sa.Single_Sim(simulation="sim_001", ROOTDIR="/path/to/sims", project="laser_shock", start_step=0, final_step=1000, step_interval=10, dump_freq=1e-9)
sim.mode_plotter()
```
